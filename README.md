Instrukcja uruchomienia

Krok	Polecenie / Czynność
1	Zainstaluj Docker Desktop ze strony docker.com
2	Przejdź do katalogu projektu: cd projekt
3	Uruchom aplikację: docker-compose up --build
4	Otwórz przeglądarkę: http://localhost:4200
5	Wybierz zakres dat i kliknij Pobierz dane
6	Filtruj dane wybierając tryb: Rok / Kwartał / Miesiąc / Dzień
